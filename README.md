# 🌐 NEXUS Infra - Autonomous Agentic AI NOC

NEXUS Infra is an Enterprise-Grade, Self-Healing Network Operations Center (NOC) designed to autonomously manage, monitor, and repair multi-vendor network infrastructures (Cisco & MikroTik) using advanced AI intent resolution.

## ✨ Core Features
- **Smart Intent Resolution:** Type commands in natural language (English/Roman Urdu/Arabic) and the AI compiles them into exact multi-vendor hardware commands.
- **Auto-Port Detection:** AI automatically resolves abstract requests (e.g., 'port 4') to precise hardware interfaces (e.g., `GigabitEthernet 1/0/4`).
- **Autonomous Watchdog & Repair:** Sentinel core actively monitors bandwidth and automatically repairs VLAN mismatches and interface faults.
- **PuTTY Live Console Overlay:** Real-time physical switch programming verified directly via console overlay.
